## Z軸寸法  
・板の厚みは2.0mm  
・魚型のアウトラインは上面から2.0mm切削（貫通）  
・ネジ頭の窪み（ø4.2mm）は上面から1.0mm切削  
・↑の中にあるネジ穴（ø2.2mm）は1.0mm切削したところから1.0mm切削（貫通）  
・ゴム足用の窪み（ø8.2mm）は上面から1.0mm切削  
・反対側も左右対称に
・板の歪みによる誤差を抑えるため、片側ずつ切削する。そのためデータは別々にする
